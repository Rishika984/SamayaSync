# SamayAsync Documentation

## Overview

SamayAsync is a web application that helps users to track their study sessions and set goals for their study plans. It is built using React, Node.js, Express, and MongoDB.

## Features

* Study Sessions: Users can track their study sessions and view their study history.
* Study Plans: Users can set goals for their study plans and track their progress.
* Weekly Progress: Users can view their weekly progress and stay motivated.

## Technologies Used

* Frontend: React, React Router, Redux
* Backend: Node.js, Express, MongoDB
* Authentication: Passport.js, JSON Web Tokens (JWT)

## How to Contribute

1. Clone the repository
2. Install the dependencies using npm install
3. Make your changes
4. Test your changes
5. Create a pull request
