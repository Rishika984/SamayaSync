# SamayaSync

SamayaSync is a study session tracker that helps students plan, record, and analyze their learning time. It helps to stay organized, stay focused, and make every minute count.

## Features

*   Create and manage study sessions
*   Set goals and track progress
*   Record study sessions with notes and tags
*   View detailed statistics of study sessions
*   Set reminders and notifications
*   Integrate with Google Calendar

## Technologies Used

*   Frontend: React, React Router, Redux, Material-UI
*   Backend: Node.js, Express, MongoDB, Passport
*   Integration: Google OAuth 2.0, Google Calendar API

## Installation

1.  Clone the repository
2.  Install dependencies with `npm install`
3.  Start the server with `npm run server`
4.  Start the client with `npm run client`

## Usage

1.  Create an account with Google OAuth 2.0
2.  Create a study session with a title, duration, and optional notes and tags
3.  Record a study session with the "Start" button
4.  View detailed statistics of study sessions with the "Statistics" button
5.  Set reminders and notifications with the "Reminders" button
6.  Integrate with Google Calendar with the "Calendar" button
